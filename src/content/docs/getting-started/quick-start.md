## 前言

SendFox 是一个面向高校社区的专业运营工具，帮助你轻松管理社区内容、提升活跃度。本文将引导你完成初始配置。

## 第一步：登录后台

1. 打开 SendFox 管理后台
2. 使用管理员分配的账号登录
3. 首次登录建议修改默认密码

## 第二步：添加平台账号

要让 SendFox 在你的社区中工作，需要绑定平台账号：

1. 进入「账号管理」页面
2. 点击「添加账号」
3. 选择目标平台（狐友 / 赞噢）
4. 根据提示完成账号绑定

<!-- 配图：账号管理页面截图 -->
<div class="doc-image-placeholder">
  <img src="/images/docs/account-management-placeholder.png" alt="账号管理页面截图" />
</div>

> **提示**：狐友平台支持手机验证码快速登录绑定，操作非常简便。

## 第三步：配置发布任务

1. 进入「任务管理」页面
2. 创建新的发布任务
3. 选择采集源和发布目标
4. 配置马甲号轮换策略（可选）
5. 启动任务

## 第四步：查看结果

任务启动后，你可以在仪表盘查看：

- 发布成功/失败的帖子数量
- 内容采集状态
- 账号健康度
