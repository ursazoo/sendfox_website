## 第一步：注册并抓包获取采集账号

### 注册

1. 打开 SendFox 管理后台，点击「注册」
2. 填写手机号，获取验证码，设置密码
3. 注册完成后登录

### 桌面客户端抓包

桌面客户端用于拦截微信小程序的网络请求，提取平台 Token。

1. 下载客户端（支持 macOS 和 Windows，Windows 推荐 Win11）
2. 用 SendFox 账号登录客户端
3. 选择采集平台（赞噢 或 逛逛）
4. 点击「生成并安装证书」：
   - **macOS**：弹出系统密码框，输入密码即可完成安装
   - **Windows**：弹出 UAC 确认框，点击「是」即可
   - 如果 macOS 自动信任失败，需在「钥匙串访问」中双击证书，将信任设为「始终信任」
5. 点击「开始抓包」，在手机微信中打开对应平台的小程序浏览内容
6. 客户端自动提取 Token 后，在「已识别 Token」列表中点击「上传」

> 抓包仅适用于微信小程序。完成后请点击「停止抓包」。

## 第二步：购买订阅并绑定学校

学校名额（Slot）决定你能绑定多少个发布学校。采集学校不受名额限制。

1. 进入「订阅」页面，选择套餐，微信扫码支付
2. 进入「学校管理」页面，点击「添加学校」
3. 添加采集学校：选择平台（赞噢/逛逛），用途设为「采集」
4. 添加发布学校：选择狐友平台，用途设为「发布」
5. 在名额列表中，将发布学校绑定到可用的 Slot

> 名额不够可额外购买学校包。解绑学校后名额释放可复用。

## 第三步：创建狐友发布账号

1. 进入「账号管理」页面，点击「狐友登录」
2. 输入手机号获取验证码，验证成功后账号自动绑定

> 马甲号功能仅限狐友圈子管理员账号，系统会自动同步马甲号列表。

## 第四步：配置 AI

1. 进入「AI 设置」页面，选择 AI 提供商，填入 API Key 并验证
2. 在「提示词管理」中新建审核提示词，定义什么内容应被屏蔽，设为审核用途并启用
3. 新建改写提示词，定义如何改写内容规避重复，设为改写用途并启用

> AI 配置全局生效，影响所有任务的内容处理。

## 第五步：创建采集和发布任务

### 采集任务

1. 进入「采集任务」页面，点击「创建任务」
2. 添加采集项：选择平台（赞噢/逛逛）→ 选择采集学校 → 选择对应账号
3. 设置采集间隔（建议 30 分钟以上），启用「自动触发 AI 审核」
4. 启动任务

### 发布任务

1. 进入「发布任务」页面，点击「创建任务」
2. 添加发布目标：选择发布学校 → 选择狐友账号 → 选择马甲号（如适用）
3. 设置发布参数：
   - **发布间隔**：5-15 分钟
   - **轮次间隔**：1-2 小时
   - **发布时段**：避开深夜
4. 开启「AI 改写」，启动任务

## 自动运行

任务启动后系统自动循环：采集内容 → AI 审核 → 改写 → 发布到目标学校 → 马甲号轮换。

你可以在「采集任务」「帖子管理」「发布任务」「账号管理」页面随时查看状态。遇到问题查看 [常见问题](/docs/faq/) 或联系客服。
