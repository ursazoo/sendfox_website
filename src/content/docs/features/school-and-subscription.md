## 学校类型

SendFox 中的学校可以配置为不同用途：

| 类型 | 说明 | 是否受名额限制 |
|------|------|----------------|
| 仅采集 | 只从该学校社区抓取内容 | 否 |
| 仅发布 | 只向该学校社区发布内容 | **是** |
| 采集 + 发布 | 同时作为采集源和发布目标 | **是**（占用发布名额） |

> 学校名额（Slot）仅影响发布学校的绑定。采集学校不受名额限制，可以自由添加。

## 学校管理

### 添加学校

1. 进入「学校管理」页面
2. 点击「添加学校」
3. 选择平台和学校
4. 设置学校用途（采集 / 发布 / 采集 + 发布）
5. 保存

### 采集-发布绑定

你可以将采集学校和发布学校建立绑定关系：

- 采集到的内容自动分配给绑定的发布学校
- 支持一对多绑定（一个采集学校对应多个发布学校）
- 未绑定的内容进入公共帖子池，所有发布任务共享

## 订阅套餐

### 名额机制

学校名额（Slot）是订阅的核心计量单位：

- 每个名额允许绑定一个发布学校
- 名额数量由订阅套餐决定
- 名额可复用：解绑学校后名额释放

### 购买订阅

1. 进入「订阅」页面
2. 选择合适的套餐
3. 微信扫码支付
4. 支付成功后名额立即到账

### 学校包

如果套餐赠送的名额不够，可以额外购买学校包：

- 在「订阅」页面点击「购买学校包」
- 学校包跟随套餐有效期

## 名额绑定流程

1. 确认已有可用的学校名额（查看「订阅」页面）
2. 进入「学校管理」页面
3. 将一个学校配置为「发布」或「采集 + 发布」
4. 在名额列表中，将该学校绑定到一个空闲的 Slot
5. 绑定完成后，该学校即可用于发布任务

> 解绑学校：在名额列表中点击「解绑」，名额释放后可重新分配给其他学校。
