## 创建发布任务

1. 进入「任务管理」→「发布任务」
2. 点击「创建任务」
3. 选择内容来源（采集内容 / 手动导入）
4. 配置发布目标（平台 + 学校）
5. 设置发布策略

## 发布策略

### 定时发布

设置发布时间间隔，系统会在指定时间段内均匀分布发布内容，避免短时间内大量发帖引起平台注意。

### 马甲号轮换

马甲号轮换功能可以自动在不同账号间切换发布，降低被识别为营销号的风险：

1. 在发布任务中选择多个发布账号
2. 配置轮换策略（顺序轮换 / 随机轮换）
3. 系统自动分配每次发布使用的账号

### 发布目标配置

每个发布账号可以独立配置发布目标学校：

- 支持绑定多个学校
- 支持跨校发布
- 自动过滤同平台学校，避免重复
- 智能编辑模式：选择已配置的账号时自动加载配置

## 发布状态跟踪

| 状态 | 说明 |
|------|------|
| 待发布 | 等待发布队列调度 |
| 发布中 | 正在执行发布 |
| 已发布 | 发布成功 |
| 失败 | 发布失败，可一键重试 |
| 暂停 | 任务已暂停 |

## 失败重试

发布失败的帖子支持一键重试，将帖子状态重置后重新进入发布队列。
