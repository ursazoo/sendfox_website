## 功能说明

除了自动采集，SendFox 还支持通过 CSV 文件批量导入帖子内容，适合需要发布自定义内容的场景。

## 导入步骤

1. 进入「帖子管理」页面
2. 点击「批量导入」
3. 下载标准 CSV 模板
4. 按照模板格式填写内容
5. 上传 CSV 文件
6. 系统自动解析并创建帖子

## CSV 模板格式

模板包含以下字段：

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| title | 帖子标题 | 是 |
| content | 帖子正文 | 是 |
| images | 图片链接（多个用逗号分隔） | 否 |
| school | 目标学校名称 | 否 |

> 图片链接支持本地路径和在线 URL。在线图片会被自动下载到云存储。

## 导入后的内容状态

批量导入的帖子会进入「待审核」状态，需要通过 AI 审核后才会进入发布队列。你也可以在帖子列表中手动批准或屏蔽。

## 注意事项

- CSV 文件编码请使用 UTF-8
- 单次导入建议不超过 100 条
- 导入前请确保内容不违反目标社区规范
- 重复导入相同标题的内容会被自动跳过
