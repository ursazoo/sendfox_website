## 手动导入帖子

当你需要发布自己准备的内容时，可以使用手动导入功能：

1. 进入「帖子管理」→「手动导入」
2. 填写帖子信息：
   - 标题
   - 正文内容
   - 上传图片
3. 选择发布账号
4. 提交导入

## Excel 批量导入

当需要导入大量帖子时，Excel 批量导入是最高效的方式：

### 第一步：下载模板

1. 点击「下载模板」按钮
2. 获取标准 Excel 模板文件
3. 模板中包含填写说明和示例

### 第二步：填写数据

按照模板格式填写帖子数据：

- **标题**：帖子标题
- **内容**：帖子正文
- **图片 URL**：可选，图片链接地址
- **来源**：标记为「手动导入」

### 第三步：上传导入

1. 点击「上传 Excel」
2. 选择填好的 Excel 文件
3. 系统自动解析并创建帖子

## 图片批量管理

上传多张图片后，可以使用拖拽方式自由分配：

1. 一次上传多张图片到图片池
2. 拖拽图片到对应的帖子卡片上
3. 支持单张帖子配多张图片
4. 未分配的图片可留待下次使用

## 失败重试

导入或发布失败的帖子可以一键重试：

- 在导入任务列表中找到失败记录
- 点击「重试」按钮
- 帖子回到导入页，修改后重新提交
