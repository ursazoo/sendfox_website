## 账号相关

### 账号显示「已过期」怎么办？

Token 有效期到期后会显示已过期。你需要重新绑定账号：
- 狐友账号：使用手机验证码重新登录
- 其他平台：重新抓包获取新的 Token

### 一个账号可以绑定多个学校吗？

可以。每个账号可以独立配置绑定的学校列表，支持跨校发布。

## 发布相关

### 发布失败的帖子怎么处理？

发布失败的帖子可以一键重试：
1. 在帖子列表中找到状态为「失败」的帖子
2. 点击重试按钮
3. 帖子会重新进入发布队列

### 什么是马甲号轮换？

马甲号轮换是在发布时自动在不同账号之间切换的功能。通过轮换发布账号，可以降低被平台识别为营销号的风险，让内容分发更加自然。

### 可以设置定时发布吗？

可以。在创建发布任务时，可以设置发布时间间隔。系统会在指定时间段内均匀分布发布内容。

## 内容相关

### AI 审核的标准是什么？

AI 审核包含以下维度：
- 敏感词检测：对照敏感词库检查
- 内容质量评估：AI 评估内容的可读性和价值
- 合规性检查：检查是否违反社区规范

审核通过的帖子才可以进入发布流程。

### 可以自定义敏感词库吗？

可以。管理员可以在「AI 配置」→「敏感词管理」中添加、删除和分类管理敏感词。

## 其他问题

### 支持哪些平台？

目前支持：
- **狐友**：支持手机号登录绑定，功能最完善
- **赞噢**：支持 Token 抓包绑定

后续会接入更多高校社区平台。

### 数据安全吗？

- 所有 Token 信息加密存储
- 数据传输使用 HTTPS 加密
- 定期安全审计和更新
