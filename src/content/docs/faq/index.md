## 账号相关

### 为什么登录时提示设备已绑定？

SendFox 目前不限制登录设备数量，你可以在多台设备上同时使用。如果遇到问题，请联系客服。

### 账号 Token 过期了怎么办？

- 通过桌面客户端重新抓包获取新 Token
- 在管理后台的账号详情页手动更新凭证
- 狐友平台可以使用手机验证码重新登录

## 抓包相关

### macOS 上证书安装后仍然无法抓包？

1. 打开「钥匙串访问」应用
2. 找到 SendFox CA 证书
3. 双击证书，展开「信任」部分
4. 将「使用此证书时」设置为「始终信任」
5. 重新启动客户端

### 抓包时客户端没有捕获到任何请求？

- 确认已点击「开始抓包」
- 确认系统代理已正确设置
- 尝试关闭并重新打开目标 App
- 检查是否有 VPN 或其他代理软件冲突

## 采集相关

### 采集到的帖子数量很少？

- 检查采集的学校是否活跃
- 确认使用的账号有权限访问目标社区
- 适当缩短采集间隔
- 查看是否有大量帖子被 AI 审核屏蔽

### 同一篇帖子被采集了多次？

正常情况下不会出现此问题。系统通过帖子的唯一 ID 进行去重。如果出现重复，可能是平台 API 返回的帖子 ID 格式发生了变化，请联系客服。

## 发布相关

### 发布任务一直不执行？

- 确认任务已启动
- 确认有「审核通过」状态的帖子可供发布
- 检查是否在设定的发布时段内

### 发布失败率很高？

- 检查账号 Token 是否过期
- 确认马甲号状态是否有效
- 检查发布间隔是否过短导致被平台限流
- 查看具体失败原因，针对性调整

### 如何避免被平台检测为营销号？

- 设置合理的发布间隔（建议 5 分钟以上）
- 开启马甲号自动轮换
- 开启 AI 改写，确保每篇内容独特
- 设置发布时段，模拟正常用户行为
- 避免单个学校短时间内发布过多内容

## 其他问题

### 如何联系技术支持？

如有以上未覆盖的问题，请通过管理后台的联系渠道与客服取得联系。
